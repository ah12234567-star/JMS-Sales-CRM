- مدير النظام: admin@jms.local / 123456
- مدير المبيعات: sales@jms.local / 123456
- ياسر الحسني: yaser@jms.local / 123456
- مندوب جدة: rep@jms.local / 123456

المزايا:
- صلاحيات مدير/مدير مبيعات/مندوب
- إنشاء وإيقاف وتفعيل المناديب
- تغيير وإعادة تعيين كلمة المرور
- العملاء وملف العميل وخيارات المتابعة
- مسارات يومية
- تنبيهات 30 يوم
- طلب تصنيع احترافي
- حاسبة وزن الحبة وعدد الحبات
- مبيعات الشهر والتحصيل ونسبة التحصيل


## شاشة الزيارات
- زيارات اليوم والفترة.
- من زار ومن لم يزر.
- عملاء أكثر من 30 يوم بدون زيارة.
- تقرير كل مندوب.
- تسجيل زيارة يدوية مع وقت الوصول والمغادرة ومدة الزيارة.

## عروض الأسعار
- إنشاء عرض سعر بواسطة المندوب.
- حالة العرض تبدأ: بانتظار اعتماد المدير.
- المدير أو مدير المبيعات يستطيع الاعتماد أو الرفض.
- لا يظهر زر الإرسال إلا بعد الاعتماد.
- إرسال العرض عبر واتساب.
- طباعة العرض / حفظه PDF من المتصفح.
- تحويل العرض المعتمد إلى طلب تصنيع.
