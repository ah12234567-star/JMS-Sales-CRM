## إصدار طلبات التصنيع
تمت إضافة حقول طلب العميل للمندوب:
- المقاس: العرض والطول ووحدة المقاس
- السماكة ووحدة السماكة
- اللون
- نوع المنتج: أكياس رول، شيت، تي شيرت، شرنك، فيلم، أكياس نفايات
- المادة: HDPE / LDPE / LLDPE / خلطة
- الطباعة وعدد ألوان الطباعة
- ملاحظات فنية


## حاسبة وزن الحبة وعدد الحبات
تمت إضافة حاسبة تلقائية داخل طلب العميل:
- يدخل المندوب العرض والطول والسماكة والخامة وكمية الطلب بالكيلو.
- يحسب النظام وزن الحبة بالجرام.
- يحسب النظام عدد الحبات المتوقع في الطلب.

معادلة وزن الحبة:
وزن الحبة بالجرام = العرض بالمتر × الطول بالمتر × السماكة بالميكرون × كثافة الخامة.

كثافات افتراضية:
HDPE = 0.95
LDPE = 0.92
LLDPE = 0.92
PP = 0.90
