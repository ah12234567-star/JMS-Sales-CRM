المزايا:

- إنشاء وإيقاف وتفعيل المناديب
- تغيير وإعادة تعيين كلمة المرور
- العملاء وملف العميل وخيارات المتابعة
- مسارات يومية
- تنبيهات 30 يوم
- طلب تصنيع احترافي
- حاسبة وزن الحبة وعدد الحبات
- مبيعات الشهر والتحصيل ونسبة التحصيل


## شاشة الزيارات
- زيارات اليوم والفترة.
- من زار ومن لم يزر.
- عملاء أكثر من 30 يوم بدون زيارة.
- تقرير كل مندوب.
- تسجيل زيارة يدوية مع وقت الوصول والمغادرة ومدة الزيارة.

## عروض الأسعار
- إنشاء عرض سعر بواسطة المندوب.
- حالة العرض تبدأ: بانتظار اعتماد المدير.
- المدير أو مدير المبيعات يستطيع الاعتماد أو الرفض.
- لا يظهر زر الإرسال إلا بعد الاعتماد.
- إرسال العرض عبر واتساب.
- طباعة العرض / حفظه PDF من المتصفح.
- تحويل العرض المعتمد إلى طلب تصنيع.

## الإصلاحات النهائية
- إصلاح صلاحيات المندوب.
- إخفاء صفحات المدير عن المندوب.
- إصلاح زر إنشاء عرض السعر.
- عرض السعر يدعم اختيار عميل موجود أو إضافة عميل جديد.
- إزالة ظهور 999 / 9999 واستبدالها بـ "لم تتم زيارته".

## ملاحظات وتقارير الزيارات
- المندوب عند الضغط على "تمت الزيارة" يفتح تقرير زيارة كامل.
- التقرير يحتوي على نوع الملاحظة، ملاحظات الزيارة، المنتج المهتم به العميل، السعر المطلوب، وموعد المتابعة.
- مدير المبيعات ومدير النظام يشاهدون كل الملاحظات في صفحة "ملاحظات الزيارات".
- الملاحظات التي تحتاج إجراء تظهر كتنبيه مفتوح.
- يمكن تحويل ملاحظة "طلب عرض سعر" إلى عرض سعر مباشرة.

## تحسين عرض السعر
- إعادة تصميم عرض السعر بالكامل بشكل احترافي.
- إضافة شعار الشركة، رقم العرض، التاريخ، تاريخ الصلاحية.
- إضافة بيانات العميل والمندوب.
- جدول منتجات مرتب.
- الإجمالي قبل الضريبة، ضريبة 15%، والإجمالي النهائي.
- شروط وملاحظات.
- خانة اعتماد مدير المبيعات وختم الشركة ورمز تحقق.
- زر طباعة / حفظ PDF وإرسال واتساب.

## تحسين اختيار العميل في عرض السعر
- استبدال القائمة الطويلة بمربع بحث.
- يمكن كتابة أي جزء من اسم العميل أو رقم الجوال أو المدينة.
- تظهر النتائج فورًا ويختار المستخدم العميل مباشرة.
- ما زال بإمكان المندوب إضافة عميل جديد من نفس شاشة العرض.
