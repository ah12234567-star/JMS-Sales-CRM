# JMS Factory CRM - Real System بدون عروض أسعار

هذه نسخة نظام حقيقي تعتمد على Supabase بدل LocalStorage.

## الموجود
- العملاء
- الزيارات وملاحظات المندوب
- طلبات التصنيع
- التحصيل والمديونيات
- المستخدمون والصلاحيات
- ظهور البيانات على اللابتوب والجوال من نفس قاعدة البيانات

## المحذوف بناء على طلبك
- عروض الأسعار
- PDF عرض السعر
- اعتماد عروض الأسعار

## طريقة التركيب
1. افتح Supabase وأنشئ مشروع جديد.
2. افتح SQL Editor.
3. انسخ ملف `supabase_schema.sql` وشغله.
4. من Supabase خذ:
   - Project URL
   - anon public key
5. افتح ملف `config.js`.
6. ضع القيم بدل:
   - PUT_YOUR_SUPABASE_URL_HERE
   - PUT_YOUR_SUPABASE_ANON_KEY_HERE
7. ارفع الملفات إلى GitHub.
8. Vercel سينشر النظام.

## عروض الأسعار مع الإلغاء
تمت إعادة عروض الأسعار مع حالة جديدة:
- المندوب ينشئ العرض.
- المدير يعتمد أو يرفض.
- بعد الإرسال للعميل تظهر للمندوب خيارات:
  - العميل وافق
  - إلغاء - العميل لم يوافق
- عند الإلغاء يجب كتابة سبب الإلغاء.
- العرض المقبول يمكن تحويله إلى طلب تصنيع.
