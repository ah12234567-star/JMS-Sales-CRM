## إصلاح ظهور عروض الأسعار في الجوال واللابتوب

سبب المشكلة السابقة أن النظام كان يحفظ البيانات داخل متصفح الجهاز فقط.
هذه النسخة تضيف مزامنة سحابية اختيارية عبر Supabase.

### طريقة التفعيل
1. افتح Supabase وأنشئ Project.
2. افتح SQL Editor.
3. انسخ محتوى ملف `supabase_schema.sql` وشغله.
4. افتح Project Settings > API.
5. انسخ:
   - Project URL
   - anon public key
6. افتح ملف `config.js`.
7. عدل القيم:
```js
window.JMS_CLOUD = {
  SUPABASE_URL: "ضع رابط Supabase هنا",
  SUPABASE_ANON_KEY: "ضع anon key هنا",
  ENABLED: true
};
```
8. ارفع الملفات إلى GitHub.
9. افتح الموقع من اللابتوب والجوال. ستظهر نفس عروض الأسعار.

إذا بقي ENABLED = false فالنظام سيعمل محليًا فقط ولن تظهر بيانات اللابتوب على الجوال.
