# JMS Sales CRM - Accounts Edition

نسخة تحتوي على:
- دخول مدير.
- إنشاء حسابات مناديب.
- ربط حساب المندوب بمندوب محدد.
- المندوب يرى عملاءه وزياراته وطلباته فقط.
- يعمل محليًا بدون Supabase، ويمكن تشغيل الجداول في Supabase لاحقًا.

الدخول الافتراضي:
- المدير: admin@jms.local / 123456
- مندوب تجربة: rep@jms.local / 123456


## Login Roles
- مدير النظام: admin@jms.local / 123456
- مندوب: rep@jms.local / 123456

واجهة الدخول تحتوي الآن على اختيار نوع الحساب: مدير النظام أو مندوب.


## إصدار طلبات التصنيع
تمت إضافة حقول طلب العميل للمندوب:
- المقاس: العرض والطول ووحدة المقاس
- السماكة ووحدة السماكة
- اللون
- نوع المنتج: أكياس رول، شيت، تي شيرت، شرنك، فيلم، أكياس نفايات
- المادة: HDPE / LDPE / LLDPE / خلطة
- الطباعة وعدد ألوان الطباعة
- ملاحظات فنية
