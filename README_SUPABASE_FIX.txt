تم تعديل app.js ليحفظ في Supabase فعليًا.

طريقة الاستخدام:
1) ارفع جميع ملفات هذه النسخة إلى GitHub بدل الملفات القديمة.
2) تأكد أن config.js موجود كما هو.
3) انتظر نشر Vercel.
4) افتح النظام واضف عميلًا جديدًا.
5) افتح Supabase > Table Editor > jms_customers > تحديث.
6) يجب أن يظهر العميل داخل الجدول.

ملاحظة:
كان الخلل أن app.js يحفظ في localStorage فقط، وتم تعديل save() وربطها بـ pushCloudData().
